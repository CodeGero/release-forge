# Contributing

Thanks for your interest!

- **Issues & feature requests:** use the templates under `.github/ISSUE_TEMPLATE`.
- **Pull requests:** open a PR against `main` using the PR template. Keep changes focused.
- **Security issues:** report privately via the Security tab — never in public issues.

This project is MIT-licensed. Premium features are distributed separately via Gumroad;
this repository contains the free, open-source tooling.
